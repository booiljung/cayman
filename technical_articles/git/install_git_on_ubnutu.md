# Install git on Ubuntu 16.04 18.04 LTS

 먼저 업데이트를 합니다.

```sh
sudo apt update
```

git을 설치 합니다.

```sh
sudo apt install git
```

git의 최초 설정을 합니다.

```sh
git config --global user.name "your name"
```

다음은 이메일을 등록합니다.

```sh
git config --global user.email "your email"
```

이제 git을 사용할 준비가 되었습니다.
